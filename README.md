<img src="" />
<!-- 실제 사용 중 사진 -->

🎨 <b>서비스 설명</b>

1. **알고리즘 질문을 등록해요**  
     >    알고리즘 문제를 풀다가 "이게 왜 틀리지?" 이런 생각을 해보신 적 있으신가요? 알고션에 질문을 등록해보세요.
2. **알고리즘 질문에 답변해요**  
     >    알고리즘 지식을 나눠요. 답변 작성 후엔 포인트를 얻을 수 있어요. 채택되면 포인트가 두 배~
3. **맘에 드는 답변을 채택해요**  
     >    맘에 드는 답변을 채택해 답변자에게 보답해요.
4. **알고리즘 실력 up!**  
     >    질문자, 답변자 모두 알고션과 함께 알고리즘 공부해요.
